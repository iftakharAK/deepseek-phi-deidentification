# Results

This folder contains lightweight reproducibility artifacts that support all
quantitative and qualitative claims reported in the paper:

"Trustworthy and Configurable PHI De-Identification in Clinical Text via Prompt-Adaptive Filtering, Explanation Generation, and Hallucination Detection."

Artifacts provided:

• Detection performance summaries (CSV)  
• Explanation quality metrics (JSON)  
• Hallucination safety metrics (CSV)  
• Training curves (CSV)  
• Qualitative synthetic case studies (text)  
• LaTeX tables used directly in the manuscript  
• Reproducibility notes

No raw datasets, clinical notes, or model checkpoints are included.

All values reported here correspond exactly to the metrics used in the final manuscript.
